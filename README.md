Este repositorio contiene el proceso de webscrapping, limpieza del conjunto de datos y análisis de datos. De manera más específica y de acuerdo con lo estipulado
en las instrucciones de los avances del proyecto:

DIAGNÓSTICO

Análisis del estado inicial de los datos: 

  *Número de registros y variables.
  
  *Tipo de dato de cada variable.
  
  *Cantidad y porcentaje de valores faltantes por variable.
  
  *Cantidad de valores únicos.
  
  *Cantidad de registros duplicados exactos.
  
  *Variables con valores fuera de dominio o inconsistentes.
  
  *Variables con formatos inconsistentes.
  
  *Identificación de problemas potenciales de calidad de datos.
  
  *Este análisis debe estar respaldado por tablas y estadísticas generadas usando código.
 

PLAN DE LIMPIEZA

Para cada variable:

  *Problemas encontrados.
  
  *Regla que utilizará para corregirlos y por qué cree que funcionará.
  
  *Riesgos asociados a la transformación.
